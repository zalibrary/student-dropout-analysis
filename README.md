# 🎓 Student Retention & Dropout Risk Analysis

## 📌 Project Overview
This project analyzes student retention and dropout patterns using academic performance and financial risk indicators. The goal is to identify high-risk student segments and provide data-driven insights for intervention strategies.

---

## 📊 Dataset Information
- Total Students: 4,424
- Target Variable: `churn_status`
  - Active
  - Retained
  - Churn (Dropout)

### Key Features:
- approval_rate
- total_units_approved
- payment_risk
- debtor
- academic_performance indicators

---

## 🔎 Analysis Performed

### 1️⃣ Exploratory Data Analysis (Python)
- Distribution of churn status
- Academic performance vs dropout
- Payment risk impact
- Correlation analysis

### 2️⃣ SQL Aggregation
- Dropout rate by financial risk
- Segment-level churn percentage
- Group-based analysis

### 3️⃣ Power BI Dashboard
Interactive dashboard covering:
- Overall dropout rate
- Academic vs financial segmentation
- Payment risk impact
- High-risk student profiling

---

## 📈 Key Insights

- Overall Dropout Rate: ~32%
- Students with low approval rate have significantly higher churn probability
- Payment risk strongly correlates with dropout
- Debtor students show elevated churn likelihood
- High-risk financial + low academic performance segment has the highest dropout concentration
